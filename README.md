# Clinic Appointments Dashboard

React TypeScript dashboard for managing clinic appointments, doctors, patients and visit statuses.

## 🚀 Demo

[Open Live Demo](https://mrssstrange.github.io/clinic-appointments-dashboard/)

---

## 🧰 Tech Stack

- React
- TypeScript
- Vite
- CSS
- LocalStorage
- GitHub Pages

---

## ✨ Features

- Create clinic appointments
- Search appointments by patient name, doctor or visit reason
- Filter appointments by status
- Filter appointments by department
- Change appointment status
- Delete appointments
- View appointment statistics
- Save data in `localStorage`
- Responsive dashboard layout

---

## 📌 Project Idea

This project simulates a clinic appointment management dashboard for medical staff.

The goal is to practice React, TypeScript, component structure, state management, forms, filters and working with frontend data.

---

## ▶️ How to Run

```bash
npm install
npm run dev
```
